Churn Prediction Project

Tools Used: Python, Pandas, Scikit-learn, Matplotlib, Seaborn
Developed a predictive machine learning model to analyze customer churn using a real-world telecom dataset. The model uses a Random Forest Classifier to predict whether a customer is likely to leave based on features like contract type, tenure, charges, and service usage. Included data cleaning, feature encoding, model training, evaluation metrics (accuracy, confusion matrix, AUC), and visual insights into feature importance.

  • Cleaned and preprocessed over 7,000 records
  • Achieved classification accuracy with an AUC score visualization
  • Identified key factors driving customer churn using feature importance
